design responsivo (ou responsividade)

Conjunto de técnicas de desenvolvimento front-end visando a contrução de sites em que o layout e a experiência geral de uso funcionem bem em dispositivos com telas de tamanhos variados

Existem duas abordagens para desenvolvimento responsivo:

mobile first (mais usado)

nesta abordagem o desenvolvimento é focado primeiro em telas pequenas, para somente depois realizarmos ajustes para telas maiores.

desktop first

nesta abordagem o desenvolvimento é focado primeiro em telas grandes, para somente depois realizarmos ajustes para telas menores.
